# MoneyMadeEasy

MoneyMadeEasy is a browser-based personal finance dashboard built with HTML, Tailwind CSS, JavaScript, Chart.js, localStorage, and GitHub Pages.

## Features

- Local profile login
- Monthly budgeting dashboard
- Income and expense tracking
- Category spending chart
- Rule-based budget insights
- Asset overview dashboard
- Emergency fund progress
- Investment portfolio tracker
- Sector allocation chart
- Monthly and all-time investment views
- Optional live stock prices with a Finnhub API key
- Export/import JSON backup

## Pages

- `index.html` - login/profile page
- `dashboard.html` - main financial dashboard
- `budgeting.html` - budgeting view
- `investments.html` - investment portfolio tracker
- `app.js` - shared utilities

## Important Notes

This is an educational demo app. Data is stored locally in the browser using localStorage and may be deleted if browser storage is cleared.

The investment tracker is for informational and educational purposes only. It is not financial advice.

## How to Use

1. Open the website.
2. Enter any profile name.
3. Add income, expenses, and investments.
4. Review dashboard insights.
5. Use Export to save a JSON backup.
